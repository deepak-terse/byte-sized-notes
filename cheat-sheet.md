## Python

### Setting up project

1. Create a virtual environment
    ```
    python3 -m venv venv
    ```

2. Activate the virtual environment
    ```
    source venv/bin/activate
    ```

3. Install dependencies
    ```
    pip3 install -r requirements.txt
    ```
    OR
    ```
    pip3 install dependency
    pip3 freeze > requirements.txt
    ```
